How to Run:

- cd to the project directory.
- compilation and linking done by:
$ make

- In order to recompile:
$ make clean
$ make

- Running the program:
$ ./create_format <FILENAME> <SIZE>
$ ./app <FILENAME>  

- Running the test (for experiments):
$ ./test
